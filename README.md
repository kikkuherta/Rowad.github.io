كشوف — نسخة مُنقّحة ومقسمة إلى ملفات

ما تم تنفيذه:
- فصل المشروع إلى ملفات: `index.html`, `css/styles.css`, `js/app.js`, `js/games.js`, `js/export.js`, `js/db-connector.js`.
- تحسين تنظيم الشيفرة وتسهيل الربط مع قاعدة بيانات لاحقاً عبر `db-connector.js`.
- تفعيل تصدير التقارير كـ CSV (يدعم Excel تلقائياً عبر BOM).
- إضافة ألعاب اختيار عشوائي بسيطة: Wheel, Scratch card, Fast train.

كيف تختبر محلياً:
1. افتح الملف `index.html` في متصفح حديث (Chrome/Edge/Firefox). لا تحتاج خادم.
2. جرِّب التبديل بين الشاشات، حفظ الدرجات، وتصدير التقارير من صفحة "التقارير".

ربط بقاعدة بيانات خارجية:
- `js/db-connector.js` يُخزن حالياً في `localStorage` ويحتوي على دالة `setRemote({url: 'https://api.example.com/save'})` و `pushToRemote()` لإرسال البيانات.
- اقتراح تنفيذ API بسيط (Node/Express) مع نقاط نهاية مثل `/api/save` و `/api/load` لاستقبال وإرجاع JSON.

خطوات مقترحة لاحقاً (قيمة مضافة):
- إضافة لوحة تفاصيل الطالب (نموذج عرض/تعديل درجة وتحويل دور مثل عريف).
- تحسين واجهة الألعاب بصرياً (canvas/animations) وحفظ النتائج.
- إضافة اختبار آلي بسيط على JS functions الرئيسة.

إذا تريد أبدأ الآن بملف خلفي (مثلاً API Express) أو أعدل لغة الـ UI أو أضيف مودال ألعاب، قل لي أي خيار تفضّل.
